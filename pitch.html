LumiStudio

每条值得留下来的信息，
最后流向了哪里？

产品判断

信息只有被决定了去处，才算真正被留下——收藏不是终点，去处才是。

核心循环

收进来 → 记一下 → 在操作台里捋清楚 → 给它一个去处

四个去处，没有第五个

放到公开页 连接到项目里 先放着 不留了

四个界面

手机 收进来，记一下，找原材料
电脑 认知操作台，把材料捋成判断
人 公开页，读判断，不读过程
Agent JSON / Markdown，结构化的同一批判断

技术实现

前端 Vite + React，单页应用，无后端、无登录
状态 四个界面共享同一份状态，不是四个孤立的静态页
数据 mock 数据 + localStorage，刷新不丢，闭环可现场演示

不是更好的收藏夹。
是一条信息真正被处理过的证明。
